# Natural Red Teaming & AI Power Unit 🌊

Changing LLM behavior through natural conversation and a little playfulness 😄 — no tools, no special prompts.

## Overview
This research explores how LLMs can be influenced through purely natural conversation,
without any hacking tools or prompt engineering techniques.

By simply talking — the way any real user would —
I identify blind spots in LLM context understanding and trigger behavioral shifts.

## Key Features
- 🗣️ Natural conversation only — no special syntax
- 🔧 Zero tools, zero prompt engineering
- 🎯 Reproduces real-world user interaction patterns
- 📝 Documented as reproducible case studies

## Target Models
Claude / ChatGPT / Gemini and other major LLMs
